# Img2PDF

Img2PDF is a simple and free image to PDF converter built for students and everyday users.

## Features

- Convert JPG, JPEG, and PNG images into one PDF
- Clean and simple interface
- Helpful articles and guides
- Student-friendly design
- Static website, easy to deploy on GitHub Pages

## Pages

- Home
- About
- FAQ
- Contact
- Privacy Policy
- Terms of Service
- Articles

## Tech Stack

- HTML
- CSS
- JavaScript
- pdf-lib

## Usage

1. Open the website
2. Select your images
3. Enter a PDF name
4. Choose a page size
5. Click convert
6. Download your PDF
